# 《幻界迴響：霓虹傳說》 - 遊戲系統企劃練習與原型展示

這是一個結合個人對戰術 RPG 體系的理解，嘗試進行系統化規劃的練習專案。我將心中對於二次元遊戲的熱忱與想法，透過互動原型的實作，試著梳理成可供開發參考的藍圖。

## 遊戲企劃核心：世界觀與定位

### 世界觀與核心理念：同一靈魂，兩種命運
本專案為二次元 RPG 新作《幻界迴響：霓虹傳說》的沉浸式宣傳站台。遊戲拒絕傳統 RPG 的坦克、補師、輸出等單一職業分類，核心圍繞**「維度人格職業系統（Dimensional Archetype System）」**展開。每位角色皆為同一靈魂於不同維度中的兩種人生，包含：
* **City Persona（現實人格）**：在星刻市維持日常與防守戰線。
* **Aether Persona（異界人格）**：在阿爾卡迪亞（Arcadia）改寫場地並解放戰術。

透過雙維度切換，引導玩家進行「信念」、「戰術」與「身分」的全面決策轉變，而非單純的數值升級。

## 核心賣點 (USPs)

### 1. 六大人格原型與 4+1 戰術框架
不採用換皮與上下位替代的數值設計，所有角色由**「六大人格原型」**（守護、意志、協調、領導、洞察、執念）出發，衍生出十二種完全互補的戰術玩法（例如：現實建立護盾 ➡️ 異界消耗護盾換取爆發）。編隊採用 4+1 陣容，考驗玩家的人格互補與次元切換時機。

### 2. 都市社交場域
不僅是虛擬空間，玩家可在都市中經營店鋪、舉辦虛擬偶像演唱會，將社交與商業化變現點（外觀、家具）深度連結，凝聚長線社群凝聚力。

### 3. 敘事級長線營運
採用章節式更新，每一週期的重大版本均會帶來全新的都市區域與異世界地圖，保持玩家的高黏著度與探索渴望。

## 進階系統設計 (Extended Design)

* **人格衝突原則 (Persona Conflict)**：每一種人格原型都形成明確且可感知的價值觀與玩法衝突（如守護原型的「保護他人 ➡️ 粉碎敵人」），徹底杜絕換皮數值與純外觀差異。
* **永續平衡數值體系**：嚴格抑制數值膨脹，透過精確的底層公式設計，確保初期角色在後續版本中仍具備功能性與搭配價值。
* **精密武裝系統**：捨棄隨機洗鍊詞條（RNG 機制），採用固定裝備屬性，將核心重心回歸戰術配置與角色技能組合。
* **智略討伐 (Multi-Solution Raids)**：Boss 戰設計至少三種破敵方案（如：弱點連鎖、場地機制、人格應用）。當階段切換時，Boss 將同步於兩個維度改寫弱點，強制引導玩家進行戰術與信念抉擇。

## 開發技術棧與工作流 (Development Workflow)

為了能將企劃邏輯具體化，我運用了現代化工具鏈進行高效開發：

* **前端架構**：Vue3 / Vuetify (完美實作 City/Aether 即時維度切換與 4+1 戰陣模擬器)。
* **AI 輔助開發**：運用 Gemini (Canvas)、Generative AI 與 AI Agent 進行規格優化與原型邏輯檢查。
* **美術設計流程**：採用精準的 Prompt Engineering，依序產出：線稿 → 上色 → 光影 → 構圖 → 角色特徵 → 五官定義，確保視覺高度統一（例如：現實黑髮 OL 形象 `Cute2.png` 與異界銀髮絕世美女 `Alter2.png` 的極致反差）。
* **審核紀錄**：開發過程包含 `Agent.md` / `Agent_v1_Enterprise.md` 審核企劃與 `OPTIMIZATION_LOG.md` 日誌追蹤。

## 專業學習與產出

* **企劃基礎**：持續練習 PRD 撰寫、雙人格衝突機制規劃、日常玩法循環 (Core Loop) 與數值模型規劃。
* **技術視角**：具備開發背景，因此能更謹慎地思考企劃需求與實作間的對接，力求產出的規格具備開發可行性，並落實於代碼層面。
* **熱忱**：長期關注傳奇網路的開發動態，對於遊戲 UI 互動與社群經營模式抱持高度學習熱忱，希望能有機會向產業前輩請益。
